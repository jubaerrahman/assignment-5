1)var is old, let is reassignable, and const is for fixed values that won't change.

2)The spread operator (...) unpacks or copies elements from arrays and objects into new ones.

3)forEach just loops, map creates a transformed array, and filter creates a something based on a condition.

4)An arrow function is a short () => syntax that's shorter than the traditional function keyword.

5)Template literals are backtick strings that let us write variables directly into text using ${}.
